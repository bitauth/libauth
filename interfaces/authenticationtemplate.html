Interface AuthenticationTemplate

An AuthenticationTemplate (A.K.A. Bitauth Template) specifies a set of locking scripts, unlocking scripts, and other information required to use a certain authentication scheme. Templates fully describe wallets and protocols in a way that can be shared between software clients.

supported

supported: AuthenticationVirtualMachineIdentifier[]
  • Defined in src/lib/template/template-types.ts:83

A list of authentication virtual machine versions supported by this template.

Virtual machine identifiers use the format CODE_YYYY_MM, where CODE is the currency code used to identify the network, and YYYY_MM is the year and month in which the specified VM version became active on the indicated network.

Identifiers with the _SPEC suffix indicate that this template is intended for compatibility with a future virtual machine version, but at the time the template was create, that virtual machine had not yet become active on the specified chain.

The earliest possible _SPEC virtual machine version is BCH_2020_11_SPEC, the first virtual machine version after the public release of the version 0 AuthenticationTemplate format.
