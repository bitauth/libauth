Optional transformed

transformed: "p2sh-locking" | "p2sh-unlocking"
  • Defined in src/lib/template/language/language-types.ts:463

The transformation type of the resulting bytecode.

Set to p2sh-locking if the resulting bytecode was transformed into a P2SH locking script (OP_HASH160 <$(<result> OP_HASH160)> OP_EQUAL).

Set to p2sh-unlocking if the resulting bytecode was transformed into a P2SH unlocking script (result <locking_script>).

This property is not defined if the result was not transformed.
